
# RSS Reader
Simple RSS Reader app demonstrating usage of various android architecture components.

## Libraries
- Kotlin
    - Coroutines
    - Flow
- Android Jetpack
    - Activity
    - Appcompact
    - Data Binding
    - Fragment
    - Hilt
    - Lifecycle
    - Navigation
    - Recycler View
    - Room
    - Work
- Third party
    - Glide
    - Moshi
    - Okhttp
    - XmlToJson
